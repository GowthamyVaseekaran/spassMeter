To start the Memory Load Plugin with the JConsole type the following text in 
a command line:

        <JDK_Home>\bin\jconsole.exe -pluginpath <Path-to-LoadPlugin>\LoadPlugin-rt.jar

Depending on Spaces in the paths you have to write the paths in quotation marks.

for example: "c:\Program Files (x86)\Java\jdk1.6.0_26\bin\jconsole.exe" -pluginpath "C:\Users\dederichs\workspace\LoadPlugin\dist\win32\LoadPlugin-rt.jar"
